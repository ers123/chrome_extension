# Chrome Web Store Listing Draft

## Title
Tab Nudge – Smart Tab Cleanup & Alerts

## Short Description (132 chars max)
Keep tabs under control with smart alerts, one-click cleanup actions, and an undo window to restore mistakes.

## Full Description
- Monitor your tab count and get notified before overload hits.
- One-click actions: close oldest, close duplicates, group by domain, or snooze alerts.
- Undo window restores recently closed tabs in seconds.
- Beautiful, accessible UI built with shadcn/ui and React.
- Dashboard view shows recent activity and usage stats.
- Privacy-first: everything runs locally; no data leaves your browser.

## Bullet Highlights
- Smart thresholds & cooldowns to prevent alert spam
- Actions popup + side panel for quick or detailed control
- Dashboard analytics stored locally (IndexedDB)
- Customizable shortcuts: Ctrl+Shift+[T/O/D/S/A]
- Works with Chrome Manifest V3
